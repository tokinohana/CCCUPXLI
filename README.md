This project is built using "Modular Monorepo" architecture. It contains several apps, each serving a specific purpose. This modularity allows for easier maintenance, development, and scalability.

### Apps:
- `user`: Handles user authentication and user management.
- `regis`: Handles registration of teams and participants.
- `ticketing`: Handles ticketing and QR code generation.
- `ccpay`: Handles payment processing for teams and participants.
